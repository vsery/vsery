## 学习 git 命令
```
git init 创建 .git文件夹
git status 查看文件状态
git add .[file] 添加文件/文件夹 . 所有文件 [添加到缓存区]
git commit -m '' 提交到本地分支.
git checkout -- [file] , 撤销 git add 货 
git diff HEAD 查看工作区和版本库里面最新版本的区别
[file] 文件名,文件夹
```
